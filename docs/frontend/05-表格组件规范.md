# 表格组件规范

## 一、基础规则

列表页必须使用 `el-table`。

表格底部必须使用 `el-pagination`。

无数据时必须使用 `el-empty`。

## 二、表格字段顺序

推荐字段顺序：

1. 序号。
2. 核心名称字段。
3. 业务关键字段。
4. 状态字段。
5. 时间字段。
6. 操作字段。

## 三、状态字段

状态字段统一使用 `el-tag`。

推荐状态映射：

| 状态 | 标签类型 |
|---|---|
| 未开始 | info |
| 进行中 | success |
| 已结束 | warning |
| 已取消 | danger |
| 待审核 | warning |
| 审核通过 | success |
| 审核驳回 | danger |
| 已签到 | success |
| 未签到 | info |
| 已签退 | success |
| 未签退 | info |

## 四、操作列规则

操作列固定在表格右侧。

常见操作：

```text
查看
编辑
进入管理
导出
删除
取消
```

如果操作数量超过 3 个，应使用“更多”收起次要操作。

危险操作必须二次确认。

## 五、分页规则

分页默认展示：

```text
total, sizes, prev, pager, next, jumper
```

分页参数建议：

```text
pageNo
pageSize
total
```
