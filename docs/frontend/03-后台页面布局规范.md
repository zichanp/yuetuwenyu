# 后台页面布局规范

## 一、列表管理页

适用于：

- 活动列表
- 报名列表
- 作品列表
- 题目列表
- 答题记录
- 投票记录
- 用户列表
- 单位列表

页面结构：

1. 页面标题区
2. 筛选查询区
3. 操作按钮区
4. 数据表格区
5. 分页区

推荐组件：

- el-card
- el-form
- el-input
- el-select
- el-date-picker
- el-button
- el-table
- el-pagination
- el-tag
- el-empty

## 二、新建 / 编辑页

适用于：

- 新建活动
- 编辑活动
- 新建题目
- 编辑题目
- 配置报名规则
- 配置评选任务
- 配置投票规则

页面结构：

1. 页面标题
2. 步骤条或分组卡片
3. 表单配置区
4. 底部操作区

推荐组件：

- el-steps
- el-card
- el-form
- el-form-item
- el-input
- el-select
- el-radio-group
- el-checkbox-group
- el-date-picker
- el-upload
- el-button
- el-alert
- el-tooltip

## 三、数据概况页

适用于：

- 活动概况
- 投稿概况
- 报名概况
- 答题概况
- 投票概况
- 单位数据概况

页面结构：

1. 活动基础信息区
2. 核心数据卡片区
3. 数据趋势区
4. 明细入口区
5. 异常提醒区
6. 空状态区

推荐组件：

- el-card
- el-row
- el-col
- el-table
- el-tabs
- el-tag
- el-empty
- el-alert
