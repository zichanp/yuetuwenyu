# 技术栈说明

## 一、项目技术栈

当前项目使用：

- Vue 3
- Element Plus

## 二、UI 组件库

后台管理页面必须统一使用 Element Plus。

基础组件优先使用 Element Plus，不允许重复开发一套新的基础 UI。

## 三、禁止重复开发的基础组件

禁止自行重复开发以下组件：

- Button
- Input
- Select
- Radio
- Checkbox
- DatePicker
- Form
- Table
- Pagination
- Dialog
- Drawer
- Tabs
- Card
- Tag
- Alert
- Tooltip
- Empty
- Message
- MessageBox

## 四、开发原则

1. 页面结构统一。
2. 组件风格统一。
3. 表单交互统一。
4. 表格展示统一。
5. 弹窗和抽屉使用规则统一。
6. 状态标签统一。
7. 空状态统一。
8. 危险操作必须二次确认。
